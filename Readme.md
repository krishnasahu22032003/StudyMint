## What is StudyMint?

StudyMint is a full-stack SaaS application that turns any topic into comprehensive, exam-ready study notes in seconds. Students enter a topic, select their class level and exam type, and receive beautifully formatted notes complete with diagrams, charts, revision summaries, and downloadable PDFs вҖ” all generated by Google Gemini AI.

It's built for students preparing for competitive exams like JEE, NEET, UPSC, and board exams вҖ” where clarity, speed, and depth of content matter most.

---

## Features

### Core Product
- **AI Note Generation** вҖ” Structured, exam-focused notes from any topic using Google Gemini
- **Revision Mode** вҖ” Condensed bullet-point summaries for quick revision before exams
- **Diagrams** вҖ” Auto-generated Mermaid.js flowcharts and concept maps embedded in notes
- **Charts** вҖ” Data visualisations rendered via Recharts for topics with numerical content
- **PDF Export** вҖ” One-click download of complete notes as a formatted PDF
- **Note History** вҖ” Full archive of all generated notes with search and filtering
- **Credit System** вҖ” Pay-as-you-go credits via Razorpay; credits never expire

### Auth & Security
- **Google OAuth** вҖ” One-click sign-in via Firebase Authentication
- **JWT** вҖ” Stateless, HttpOnly cookie-based sessions with refresh token rotation
- **Zod Validation** вҖ” End-to-end runtime schema validation on all API inputs
- **Signature Verification** вҖ” Razorpay webhook signature verification on every payment

### UX & Design
- **Dark / Light Mode** вҖ” Full theme support via `data-theme` attribute and CSS custom properties
- **Framer Motion** вҖ” Fluid page transitions, staggered list animations, and micro-interactions
- **Fully Responsive** вҖ” Mobile-first layout; collapsible sidebar on all screen sizes
- **Skeleton Loaders** вҖ” Pulsing placeholders during async states for zero layout shift

## Getting Started

### Prerequisites

- Node.js вүҘ 20
- MongoDB Atlas cluster (or local MongoDB)
- Google Cloud project with Gemini API enabled
- Firebase project with Google OAuth enabled
- Razorpay account

### 1. Clone the repository

```bash
git clone https://github.com/krishnasahu/studymint.git
cd studymint
```

### 2. Set up the server

```bash
cd server
npm install
cp .env.example .env
```

Edit `.env` with your credentials (see [Environment Variables](#environment-variables)), then:

```bash
npm run dev
```

Server runs on `http://localhost:5000`.

### 3. Set up the client

```bash
cd client
npm install
cp .env.example .env
```

Edit `.env`, then:

```bash
npm run dev
```

Client runs on `http://localhost:5173`.

## How It Works

```
User enters topic
      в”Ӯ
      в–ј
Client sends request with topic, class level, exam type, options
      в”Ӯ
      в–ј
Server validates input with Zod вҶ’ checks JWT вҶ’ checks credit balance
      в”Ӯ
      в–ј
Gemini API generates structured note content (markdown + mermaid + chart data)
      в”Ӯ
      в–ј
Note saved to MongoDB вҶ’ credits deducted вҶ’ response streamed to client
      в”Ӯ
      в–ј
FinalResult component renders markdown, Mermaid diagrams, Recharts visualisations
      в”Ӯ
      в–ј
User can download as PDF or revisit from Note History
```

## Payment Flow

```
User selects credit plan вҶ’ clicks Buy Now
      в”Ӯ
      в–ј
POST /api/credit/order  вҶ’  Razorpay order created  вҶ’  checkout URL returned
      в”Ӯ
      в–ј
User completes payment on Razorpay hosted page
      в”Ӯ
      в–ј
POST /api/credit/verify  вҶ’  HMAC signature verified  вҶ’  credits added to user
```

All payment verification uses Razorpay's HMAC-SHA256 signature to ensure no client-side tampering is possible.

## API Reference

### Auth
| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/auth/google` | Firebase Google OAuth sign-in |
| `POST` | `/api/auth/signout` | Clear session cookies |
| `GET` | `/api/auth/me` | Get authenticated user |

### Notes
| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/notes/generate` | Generate AI note (costs credits) |
| `GET` | `/api/notes` | Get all user notes |
| `GET` | `/api/notes/:id` | Get single note by ID |

### Credits & Payments
| Method | Endpoint | Description |
|---|---|---|
| `POST` | `/api/credit/order` | Create Razorpay order |
| `POST` | `/api/credit/verify` | Verify payment + add credits |

All protected routes require a valid JWT passed as an HttpOnly cookie.
